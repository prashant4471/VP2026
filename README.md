# Valentine SPA (Next.js)

A single-page Valentine's Day experience with a romantic UI, interactive surprise modal, and sections for love notes, timeline memories, gallery placeholders, and promises.

## Getting Started

Install dependencies (already done if you ran create-next-app) and start the dev server:

```bash
npm run dev
```

Open http://localhost:3000 to view the app.

## Customize

- Edit the content in [src/app/page.tsx](src/app/page.tsx)
- Adjust styling in [src/app/globals.css](src/app/globals.css)
- Replace gallery placeholders with real images

## Build

```bash
npm run build
npm run start
```
